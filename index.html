game.js by felipeelias

A javascript framework to create HTML games

Play pong in Javascript

Start game: 'space'
Reset: 'R'
Player 1 (left): 'W': move up, 'S' move down
Player 2 (right): 'Up arrow': move up, 'Down arrow': move down
